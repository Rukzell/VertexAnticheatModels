# How can I use one of these models?
It's simple! Create a models folder in the server's root directory and add aim_detection_model.json to it(available in 26.2.2).

# How can I train my own model?
Find the RotationDumper.java(VertexAnticheat src) file and you will understand everything.

# How to retrain a model?
